Programa que calcula un sismograma de un pulso de Ricker y mostrar la salida en formato texto

Los fuentes de este proyecto puede descargarse de GitHub mediante
    $ git clone git://github.com/antoniocarrillo69/Sismograma.git
Para compilar usar:
    $ g++ *.cpp
Ejecutar mediante:
    $ ./a.out
